To enable Deepgram real-time transcription with speaker diarization:

1. Create a .env.local file in the application directory
2. Add your Deepgram API key:

DEEPGRAM_API_KEY=your_deepgram_api_key_here

3. Get your API key from: https://console.deepgram.com/
4. Restart your development server: npm run dev

The transcription feature will be automatically enabled when the API key is detected.
